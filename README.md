# 心理陪伴AI (mental-helper)
基于本地大模型的离线心理陪伴AI聊天机器人，可打包分发，纯Python实现

## 项目简介
本项目是一个基于本地大语言模型的离线心理陪伴对话系统，无需联网即可使用，保护用户隐私。
独立完成模型部署、对话逻辑实现、GUI界面开发，并将项目完整打包为可分发的exe安装包，用户双击即可安装使用，无需配置环境与模型。

## 功能特性
- 本地运行大语言模型 qwen2.5:0.5b，纯离线使用
- 完整对话交互界面，支持上下文理解
- 可一键打包为exe安装包，一键分发
- 无需Python环境、无需Ollama环境、无需手动下载模型

## 技术栈
- Python
- Ollama 本地大模型部署
- Tkinter 图形界面
- PyInstaller 打包为exe
- Inno Setup 制作安装包

## 使用说明
1. 下载Release中的安装包
2. 双击运行安装程序，一路点击Next完成安装
3. 安装完成后自动启动程序，直接开始聊天
